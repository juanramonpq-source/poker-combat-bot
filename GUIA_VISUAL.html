⚙️ EDITOR VISUAL DE POSICIONES

Guía visual para PoCoBOT - Ruta Ceniza

🚀 Instalación (Primera vez)

1

Instalar Live Server en VS Code

Ve a Extensions (Ctrl+Shift+X) → Busca "Live Server" → Instala de Ritwick Dey

2

Verificar archivos en la carpeta

Asegúrate que tienes:

  • ✓ pocobot_mapa_interactivo.html
  • ✓ EDITOR_VISUAL.js

Ambos deben estar en la carpeta: Poker Combat-bot

3

Abrir con Live Server

Haz clic derecho en pocobot_mapa_interactivo.html → "Open with Live Server"

El navegador se abre automáticamente en http://localhost:5500/

🎮 Usar el Editor Visual

Presiona E para activar el editor

Verás un panel en la esquina inferior derecha

Arrastra los nodos con el ratón

Las posiciones se actualizan en tiempo real

Observa las coordenadas en el panel

Muestra X% e Y% de cada nodo

Haz clic 📋 COPIAR TODO

El código se copia al portapapeles automáticamente

💾 Guardar Cambios Permanentes

Para que los cambios sean permanentes:

1

Abre el archivo HTML en VS Code

Archivo: pocobot_mapa_interactivo.html

2

Busca la línea const nodes = [

Usa Ctrl+F para buscar rápidamente

3

Reemplaza TODO el array

Selecciona desde const nodes = [ hasta ];

Pega lo que copiaste del editor

4

Guarda el archivo

Ctrl+S (Windows/Linux) o Cmd+S (Mac)

5

El navegador se recarga automáticamente

✓ Los cambios están guardados

⌨️ Atajos de Teclado

Tecla Acción
E Activar/Desactivar Modo Editor
G Mostrar/Ocultar Grid (en modo editor)
R Resetear posiciones (en modo editor)
💡 Tip: El Grid es útil para alinear nodos en posiciones regulares (múltiplos de 10%)

⚡ Flujo Rápido (4 pasos)

  1. Abrir: pocobot_mapa_interactivo.html con Live Server
  2. Editar: Presiona E + arrastra nodos
  3. Copiar: Haz clic en 📋 COPIAR TODO
  4. Guardar: Reemplaza el array en el HTML y presiona Ctrl+S
✓ Los cambios se guardan permanentemente cuando guardas el archivo HTML

🐛 Solución de Problemas

❌ No veo el panel del editor
→ Presiona E nuevamente
→ Abre la consola (F12) y busca errores
❌ Los nodos no se mueven
→ Verifica estar en Modo Editor (panel visible)
→ Intenta hacer clic primero en un nodo
❌ El código no se copiar
→ Abre la consola (F12 → Consola)
→ El código también aparecerá impreso allí
❌ Los cambios no se guardan
→ Asegúrate de haber presionado Ctrl+S
→ Recarga la página con F5

📚 Información Adicional

¿Cómo funcionan las coordenadas?

  • X: Posición horizontal (0% = izquierda, 100% = derecha)
  • Y: Posición vertical (0% = arriba, 100% = abajo)

¿Puedo ver el Grid?

Sí, presiona G en modo editor para ver una malla de 10% que te ayuda a alinear.

¿Se pierden los cambios si recargo?

No, siempre que hayas guardado el archivo HTML. Live Server se encarga de recargar automáticamente.
