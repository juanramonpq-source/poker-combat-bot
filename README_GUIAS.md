## 🔗 REFERENCIAS RÁPIDAS

- Comando: "Open with Live Server"
- Puerto: 5500 por defecto

### PHOENIX CODE
- Preview integrado (sin extensiones)
- Atajo: Click derecho → "Preview"
- Terminal: `Ctrl+` ` (tilde)
- Servidor: `python3 -m http.server 8000`

- Grid: Presiona **G**
- Resetear: Presiona **R**

## 🆘 APOYO Y TROUBLESHOOTING

### Problemas comunes

**"No veo el panel del editor"**
→ Presiona E nuevamente, verifica consola (F12)

**"El archivo no se guarda"**
→ Usa Ctrl+S, verifica permisos de archivo

**"Los cambios no se ven"**
→ Recarga la página (F5), borra cache (Ctrl+Shift+Del)

**"La página carga lenta"**
→ Cierra el grid (G), cierra otras pestañas

### Buscar soluciones
1. Primera opción: Lee "Troubleshooting" en tu guía
2. Segunda opción: Abre consola (F12) y busca errores
3. Tercera opción: Lee COMPARATIVA_EDITORES.md
